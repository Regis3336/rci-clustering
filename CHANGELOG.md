# Changelog

All notable changes to **RCI: Reverse Clustering Impact** are documented here.  
This project follows a structural-behavioral versioning logic rather than semantic versioning.

## [1.2.0] — README & Visual Communication Update — 2025-03-30

### Added
- **Hero comparison image (`images/hero_comparison.png`):**  
  Visual benchmark showing KMeans, DBSCAN, and RCI on Torus, Trefoil, and Dumbbell datasets side by side. Generated by the new standalone script `generate_hero_image.py`.
- **`generate_hero_image.py`:**  
  Standalone script to reproduce `images/hero_comparison.png`. Attempts to use `SpectralRCI` directly; falls back to a self-contained geometric approximation if the module is unavailable.

### Changed
- **README fully restructured:**
  - Hook rewritten to lead with the failure modes of classical methods before introducing RCI.
  - Hero image promoted to top of document, immediately after the hook.
  - Quick Start and minimal example moved above theoretical exposition.
  - Overview compressed — three core contributions (Curvature Law, Matrix-on-Demand, MEI) presented concisely without loss of technical content.
  - Operational Coherence section repositioned after benchmark results and reframed as a statement of confidence rather than a defensive disclaimer.
  - DOI badge added to header.
  - `fit_predict` corrected to `fit` + `predict` in the minimal example, matching the actual `SpectralRCI` API.
